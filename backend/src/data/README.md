# data/

Mock JSON "database" for AeroTwin.

Populated in **Phase 3** with:
- `regions.json` — the 18 simulated flood-prone regions (North/South/East/West India)
- `hazards.json` — generated hazard datasets per region (flood zones, power
  infrastructure, towers, buildings, no-fly zones, relief camps, hospitals,
  safe landing zones)
- `missions.json` — mission history log used by `GET /missions` (Phase 5)

All data here is fictional and generated for demonstration purposes only.
